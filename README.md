# Nexus AI Chat

Real-time AI chat interface with streaming responses, conversation history, and multi-model support. Built with Next.js, Vercel AI SDK, and TypeScript.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Components:** shadcn/ui
- **Charts:** Recharts
- **Tables:** TanStack Table
- **Animations:** Framer Motion

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
